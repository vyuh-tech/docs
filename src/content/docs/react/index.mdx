## Introduction

Vyuh for React brings the powerful CMS-driven approach of the Vyuh framework to
the React ecosystem. It enables you to build modular, scalable React
applications with dynamic content updates powered by headless CMS platforms.

### Key Features

- **CMS-driven UI development** - Connect your React apps to Sanity.io and other
  headless CMS platforms
- **Dynamic content updates** without code changes or redeployments
- **Modular architecture** for building scalable applications with composable
  features
- **Live content editing** for real-time preview of CMS changes
- **Extensible plugin system** for integrating with third-party services
- **Standardized patterns** for content types, layouts, and routing

### Why Vyuh for React?

Vyuh for React bridges the gap between developers and content creators. It gives
your team the ability to update content, layouts, and user journeys without
requiring code changes or new deployments. This approach dramatically reduces
development time and allows for rapid experimentation with different content
variations.
